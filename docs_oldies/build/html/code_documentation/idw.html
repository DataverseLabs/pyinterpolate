idw module¶

Available functions:

  • inverse_distance_weighting: performs Inverse Distance Weighting with a given set of points and an unknown location.

inverse_distance_weighting()¶

pyinterpolate.idw.inverse_distance_weighting(
    known_points,
    unknown_location,
    number_of_neighbours=-1,
    power=2.)

INPUT:

  • known_points: (numpy array) array of coordinates and their values,

  • unknown_location: (numpy array) pair of coordinates,

  • number_of_neighbours: (int) default=-1 which means that all known points will be used to estimate value at the unknown location. Can be any number within the limits [2, length(known_points)],

  • max_range: (power) default=2, power value must be larger or equal to 0, controls weight assigned to each known point.

OUTPUT:

  • (float) value at unknown location.
